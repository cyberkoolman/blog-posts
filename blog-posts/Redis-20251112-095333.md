### Code Example: Redis Key-Value Storage

To demonstrate how effortless Redis can be, here's a basic example in Python using the `redis` library:

```python
import redis

# Connect to Redis server
r = redis.Redis(host='localhost', port=6379, db=0)

# Set a key-value pair
r.set('example_key', 'Hello, Redis!')

# Retrieve the value of a key
value = r.get('example_key')
print(value.decode('utf-8'))  # Output: Hello, Redis!
```

With just a few lines of code, you can store and retrieve data using Redis.
